1 Introducción

En esta semana comenzamos el estudio del cálculo integral, la operación inversa a la derivación. Exploraremos el concepto de antiderivada o función primitiva, que nos permite reconstruir una función a partir de su derivada, y presentaremos la integral indefinida como la familia de todas las primitivas.

2 Concepto de Antiderivada o Primitiva

Una función \(F\) se llama antiderivada (o primitiva) de una función \(f\) en un intervalo \(I\) si:

\[ F'(x) = f(x) \quad \text{para todo } x \in I \]

2.1 Relación entre Derivada e Integral

La derivación y la integración son operaciones inversas: - Si derivamos una función y luego integramos, recuperamos la función original (salvo una constante). - Si integramos una derivada, obtenemos la función original más una constante.

3 Integral Indefinida

La integral indefinida de una función \(f\) se denota como:

\[ \int f(x) \, dx = F(x) + C \]

donde: - \(f(x)\) es el integrando - \(F(x)\) es una primitiva de \(f(x)\) - \(C\) es la constante de integración (representa todas las primitivas posibles)

3.1 Propiedades Básicas

  1. \(\int k \, dx = kx + C\) (constante)
  2. \(\int x^n \, dx = \frac{x^{n+1}}{n+1} + C\), para \(n \neq -1\)
  3. \(\int k \cdot f(x) \, dx = k \int f(x) \, dx\)
  4. \(\int [f(x) \pm g(x)] \, dx = \int f(x) \, dx \pm \int g(x) \, dx\)

4 Laboratorio

4.1 Relación entre Derivada e Integral

Ejemplo 1: Verificar que \(F(x) = x^3 + 5\) es una primitiva de \(f(x) = 3x^2\).

  1. Derivamos \(F(x)\): \(F'(x) = 3x^2\).
  2. Como \(F'(x) = f(x)\), entonces \(F(x)\) es una primitiva de \(f(x)\).

Ejemplo 2: Encontrar tres primitivas diferentes de \(f(x) = \cos x\).

  • \(F_1(x) = \sin x\) (pues \(\frac{d}{dx}[\sin x] = \cos x\))
  • \(F_2(x) = \sin x + 5\) (pues la derivada de la constante es 0)
  • \(F_3(x) = \sin x - 2\) (pues la derivada de la constante es 0)

Todas las primitivas se escriben como: \(\int \cos x \, dx = \sin x + C\).

4.2 Cálculo de Integrales Indefinidas Básicas

Ejemplo 3: Calcular \(\int 5 \, dx\).

\[ \int 5 \, dx = 5x + C \]

Ejemplo 4: Calcular \(\int x^4 \, dx\).

\[ \int x^4 \, dx = \frac{x^{4+1}}{4+1} + C = \frac{x^5}{5} + C \]

Ejemplo 5: Calcular \(\int (3x^2 - 2x + 1) \, dx\).

\[ \int (3x^2 - 2x + 1) \, dx = 3 \cdot \frac{x^3}{3} - 2 \cdot \frac{x^2}{2} + x + C = x^3 - x^2 + x + C \]

Ejemplo 6: Calcular \(\int \frac{1}{x^2} \, dx\).

Reescribimos: \(\frac{1}{x^2} = x^{-2}\).

\[ \int x^{-2} \, dx = \frac{x^{-2+1}}{-2+1} + C = \frac{x^{-1}}{-1} + C = -\frac{1}{x} + C \]

Ejemplo 7: Calcular \(\int \sqrt{x} \, dx\).

Reescribimos: \(\sqrt{x} = x^{1/2}\).

\[ \int x^{1/2} \, dx = \frac{x^{1/2+1}}{1/2+1} + C = \frac{x^{3/2}}{3/2} + C = \frac{2}{3} x^{3/2} + C \]

Ejemplo 8: Calcular \(\int (x^3 - 2x + 5) \, dx\).

\[ \int (x^3 - 2x + 5) \, dx = \frac{x^4}{4} - 2 \cdot \frac{x^2}{2} + 5x + C = \frac{x^4}{4} - x^2 + 5x + C \]

Ejemplo 9: Encontrar \(f(x)\) si \(f'(x) = 6x^2 - 4x + 3\) y \(f(1) = 5\).

  1. Integramos para encontrar \(f(x)\): \[ f(x) = \int (6x^2 - 4x + 3) \, dx = 6 \cdot \frac{x^3}{3} - 4 \cdot \frac{x^2}{2} + 3x + C = 2x^3 - 2x^2 + 3x + C \]
  2. Usamos la condición \(f(1) = 5\): \[ 2(1)^3 - 2(1)^2 + 3(1) + C = 5 \implies 2 - 2 + 3 + C = 5 \implies 3 + C = 5 \implies C = 2 \]
  3. Por lo tanto: \[ f(x) = 2x^3 - 2x^2 + 3x + 2 \]

Ejemplo 10 (Situación problema): La velocidad de una partícula (en m/s) está dada por \(v(t) = 4t - 2\). Si en \(t = 0\) la partícula está en la posición \(s(0) = 3\) m, encuentra la función de posición \(s(t)\).

  1. Recordemos que \(v(t) = s'(t)\), por lo tanto \(s(t) = \int v(t) \, dt\).
  2. Integramos: \[ s(t) = \int (4t - 2) \, dt = 4 \cdot \frac{t^2}{2} - 2t + C = 2t^2 - 2t + C \]
  3. Usamos la condición \(s(0) = 3\): \[ 2(0)^2 - 2(0) + C = 3 \implies C = 3 \]
  4. Por lo tanto: \[ s(t) = 2t^2 - 2t + 3 \]

5 Ejercicios Propuestos

Calcula las siguientes integrales indefinidas:

  1. \(\int 7 \, dx\)
  2. \(\int x^8 \, dx\)
  3. \(\int (4x^3 - 6x) \, dx\)
  4. \(\int \frac{1}{x^3} \, dx\) (Reescribe como potencia)
  5. \(\int \sqrt[3]{x} \, dx\)
  6. \(\int (5t^4 - 3t^2 + 2t - 1) \, dt\)

Encuentra la función \(f(x)\) que satisface las condiciones dadas:

  1. \(f'(x) = 2x - 5\), \(f(2) = 1\)
  2. \(f'(x) = \sqrt{x}\), \(f(4) = 10\)

Problema de aplicación:

  1. La aceleración de un objeto es \(a(t) = 6t\) m/s². Si la velocidad inicial es \(v(0) = 2\) m/s y la posición inicial es \(s(0) = 5\) m, encuentra las funciones \(v(t)\) y \(s(t)\).

  2. Verifica que \(F(x) = e^{2x} + 3\) es una primitiva de \(f(x) = 2e^{2x}\).

6 Conclusión

Hemos introducido la integral indefinida como la operación inversa de la derivación. La relación entre derivada e integral es fundamental para el cálculo. En la próxima semana, extenderemos este concepto a la integral definida y al área bajo la curva.
